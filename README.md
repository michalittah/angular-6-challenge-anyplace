

#Anyplace - Angular 6 Challenge


### Quick start
```bash
# clone repo
# --depth 1 removes all but one .git commit history
git clone --depth 1 git@github.com:michalittah/angular-6-challenge-anyplace.git

# install the repo with npm
npm install

# start the server
npm start

```


### build files
```bash
# development
npm run build:dev
# production (jit)
npm run build:prod
# AoT
npm run build:aot
```
